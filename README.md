# ArtPrompter

ArtPrompter is a desktop app that generates art prompt phrases using customizable prefixes, subjects, and suffixes.

## Supported operating systems

- Windows
- macOS
- Linux

## Installation

1. Clone the repository:
   - `git clone https://github.com/AshleyDavage/ArtPrompter.git`
2. Build and run:
   - `dotnet build`
   - `dotnet run --project ArtPrompter/ArtPrompter.csproj`
